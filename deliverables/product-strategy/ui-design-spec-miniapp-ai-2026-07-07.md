```

├─ pages/photo/              # 证件照
│   ├─ index                 # 照片选择
│   ├─ edit                  # 编辑工具流（核心）

│   └─ export                # 导出（规格/回执）
├─ pages/resume/             # 简历

│   ├─ editor                # 元素级编辑器
│   ├─ optimize              # AI 润色预览

│   └─ index                 # 向导式 step-by-step
└─ pages/me/                 # 我的

```

### 2. 证件照编辑流（pages/photo/edit）⭐ 核心

**导航**：顶部步骤条 "步 2/4：编辑"

**照片预览区**：占屏幕高度 40%，背景 `--c-bg-surface`，居中显示当前处理结果。底部圆点指示器标记当前步骤。

**工具栏（横排滚动 Chip）**：抠图 → 去背景 → 去水印 → 修图 → 检测 → 导出

每个 Tab 切换时，照片预览区即时刷新。当前激活 Tab 绿色高亮。

**各 Tab 参数面板**：

| Tab | 参数 | UI 组件 |
|-----|------|--------|
| 抠图 | 精度（标准/增强） | 双选按钮组 |
| 去背景 | 颜色（白/蓝/红/浅蓝/浅灰） | 5 色圆形色块点击 |
| | 自定义背景 | 上传图片按钮 |
| 去水印 | 涂抹区域 | 照片上手指绘制 mask（Canvas） |
| | 消除强度 | 滑块 0-100 |
| 修图 | 亮度/对比度/饱和度/色温/锐度 | 5 个滑块，-100~+100 |
| | 磨皮 | 滑块 0-100 |
| 检测 | 手动触发检测 | 按钮 + 结果列表 |
| 导出 | 规格（1寸/2寸/护照...） | 下拉选择器 |
| | 回执（城市/数量） | 开关 + 下拉 |

**底部操作栏**：主按钮（如"开始抠图""确认换底""开始检测"）+ 重置按钮（恢复原始照片）。

**顶部**：通过/不通过大标记（绿色对勾 or 红色叉号）

```
┌─────────────────────────────────┐
│  □ 光照检测    ● 80分  ✅ 通过   │
│  □ 姿态检测    ● 45分  ❌ 未通过 │
│      建议：头部略微右偏，请正对   │
│      相机重新拍摄               │
│  □ 遮挡检测    ● 95分  ✅ 通过   │
│  □ 背景检测    ● 70分  ✅ 通过   │
│  □ 尺寸检测    ● 100分 ✅ 通过   │
│  □ 人脸检测    ● 90分  ✅ 通过   │

```

**底部**："重新拍照" / "使用当前照片" 按钮。

### 4. 简历导入（pages/resume/import）

**顶部**：标题 "导入简历" + 步骤条

**上传区**：虚线框拖拽/点击区域，支持 4 种格式：

```
┌─────────────────────────────────┐
│                                 │
│           [上传图标]             │
│        点击上传简历文件          │
│                                 │
│   支持 PDF · Word · 图片 · 文本  │
│                                 │
├─────────────────────────────────┤
│  [粘贴文本]  或直接粘贴简历内容    │
│  ┌─────────────────────────────┐│
│  │                             ││
│  └─────────────────────────────┘│
└─────────────────────────────────┘
```

**AI 解析**：上传后显示进度条 "AI 正在解析简历..." → 解析完成后显示置信度 + 字段列表（逐字段标注来源 "AI" / "直接"）。

---

### 5. 简历模板选择（pages/resume/template）

**3×2 网格**：5 套模板 + "更多"（P2 预留）

每格：模板缩略图（显示排版风格示意）+ 名称

当前选中：绿色边框 1.5px + 勾选标记。

**底部**："预览" / "下一步：编辑" 按钮。

---

### 6. 简历元素级编辑器（pages/resume/editor）

**预览区**：实时预览，点击元素 → 选中态（浅蓝边框 1px）

**底部属性面板**（弹起 Sheet）：

```
┌─────────────────────────────────┐
│  [内容编辑区 - 可编辑文本]        │
│                                 │
│  字体大小: [━━━━━●━━] 14px      │
│  字体粗细: [普通]/粗体            │
│  对齐方式: ← → ≡               │
│                                 │
│  [保存] [取消]                  │
└─────────────────────────────────┘
```

**交互**：点击 preview 中的 block → 弹起属性 Sheet → 修改文字/样式 → 确认 → 预览即时刷新。

### 8. 导出页（pages/photo/export & pages/resume/preview）

**证件照导出**：
- 规格选择器（下拉）
- 是否含回执（开关）
- 导出按钮 → 弹出 "保存到相册" / "分享给朋友"

**简历导出**：
- PDF/Word 双按钮
- 显示预估文件大小
- 导出完成后 → 预览 + "转发给 HR"（wx.shareFileMessage）

---

### 9. 分享卡片

小程序 `onShareAppMessage` 配置：

```js
{
  title: `${userName} 的简历`,
  path: `/pages/resume/preview?id=${resumeId}`,
  imageUrl: `cloud://share-card-${resumeId}.png`  // 云存储生成的分享图
}
```

**分享图生成**（云函数 `photoExport` 附带产出）：证件照 + 姓名/职位 + "AI 求职一站式" 标识 + 小程序码，1600×900 竖版。

## 交互规范

### 加载态

| 场景 | 反馈 |
|------|------|
| AI 生成职业照 | 进度条 + "AI 正在为您生成职业照..." |
| AI 解析简历 | 骨架屏（shimmer） + "AI 正在分析..." |
| 抠图 | 照片区域 pulsating opacity + "正在提取人像..." |
| inpainting 去水印 | 进度 0-100% + "AI 正在填充..." |
| PDF 导出 | spinner + "正在生成 PDF..." |

### 错误态

| 错误 | UI |
|------|----|
| 额度耗尽 | 全屏插屏 "生图额度已用完" + 观看广告按钮 |
| AI 服务不可用 | Toast "稍后再试" + 重试按钮 |
| 内容安全拦截 | Toast "照片含违规内容" + 返回 |
| 合规检测不通过 | 红色叉号 + 具体修正指引 |

### 空态

- 历史记录空：插图 + "还没有制作过简历" + "开始制作" 按钮
- 模板加载中：3 个骨架卡片

### 确认/破坏操作

- 重置照片 → 弹窗确认 "确认放弃当前编辑？"
- 删除简历 → 弹窗确认 "确认删除？"
- 退出工作流 → 提示 "您的进度已保存，可随时继续"

---

## 无障碍

- 所有按钮有 `aria-label`
- 滑块有实数值读屏支持
- 合规检测不通过项使用 `role="alert"` 播报
- 色彩对比度 ≥ 4.5:1（文字/背景）
