1. 产品定义
⼀句话描述：⾯向国内微信⽤⼾的「AI ⼯具串联⼯作流」⼩程序，把证件照、简历、PDF、分享在⼩程序内串成向导式流⽔
线，⽤免费混元做 AI 增强，⼀站式闭环、微信原⽣分享。
⽬标⽤⼾：国内微信⽤⼾——求职者（应届/社招）、办事考证⼈群。与 web 端（Craftisle）完全割裂。
核⼼差异化：市场上⽆「证件照+简历+导出」⼀站式串联产品（竞品均为单点）。我们的特点=串联⼯作流 + 元素级可编辑模
板 + 多格式导⼊ AI ⾃动映射 + 双⼊⼝ + 微信原⽣分享 + 免费混元降本。

4. ⽤⼾场景
A 纯证件照：⾸⻚→证件照→类型/城市/底⾊→⽣成Hy3→检测→换底→导出/回执 / 或「去做简历」带⼊照⽚。
B 完整求职流：⾸⻚→求职简历→选模板5套→①出照→②导⼊⽂件/⼿填+AI填充→③导PDF→④分享。
C 双⼊⼝续接：证件照「去做简历」→选模板带照⽚→导⼊填充→导出分享。
5. 功能详细设计
5.1 证件照（F2/F3/F4/F5）
规格：⼀⼨/⼆⼨/⼩⼀⼨/⼩⼆⼨、教资、国考省考、驾驶证、签证等；底⾊红/蓝/⽩；DPI 300‒1000；回执按城市规则。
⽣成（F2）：场景词/参考→ai.generatePortraitPromptHy3→ai.generateImageHyImage3.0，2–4张。
合规检测（F3）：优先评估第三⽅开源引擎（⼈脸检测/姿态/光线/遮挡/背景/尺⼨合规类开源项⽬），评估通过则接⼊；⽆合适
开源则⾃研规则引擎。该能⼒⾮混元免费额度，成本独⽴评估。photo.detectCompliance 返回逐项结果+修正指引。
换底/美化/导出（F4）：发丝级抠图（依赖引擎）、底⾊切换、轻度美化、回执⽂本、导出。
双⼊⼝（F5）：「去做简历」带⼊照⽚资产。
5.2 简历模板系统（F6/F6b/F6c）
通⽤模板（F6）：⾸版 5 套国内适⽤版式（简约/商务/创意/学术/应届），结构含基本信息证件照位/教育/⼯作实习/项⽬/技
能/⾃评/荣誉。元素带语义 tag。
元素级可编辑：⽂本/标题/列表/图⽚/分隔线/留⽩，⽀持内容改、位置拖拽、样式字体/字号/⾊/粗、显隐、增删。所⻅即
所得。
基于模板创建：选模板→实例化，改实例不影响原模板。
导⼊+AI填充（F6b）：⽀持 PDF/Word.docx/图⽚OCR/⽂本 → resume.importParseHy3 → resume.applyToTemplate
按tag⾃动映射填充，缺失留空提⽰、多余进”其他”。
AI润⾊（F6c）：粘贴 JD → resume.optimizeHy3 做 STAR 改写 + 量化补充 + JD 匹配，保留真实经历、禁虚构、输出亮点
+匹配度。
5.3 导出与分享（F7/F7b/F8）
PDF 导出（P0）：基于模板实例，A4 中⽂排版，嵌⼊证件照。
Word 导出（P1）：resume.exportWord，对标竞品免费 Word。
分享（F8）：转发好友/朋友圈卡⽚/存相册。

created → step_template(selected) → step1_photo(idle→pending→success/failed, 或从证件照带⼊)

→ step3_pdf(idle→pending(export)→success/failed) → step4_share(idle→shared→completed)
任⼀步 failed 可重试不丢数据；可暂停，从「我的」续接。

13. 边界与异常
异常 处理
⽹络失败 重试+保留数据
额度耗尽/过期 提⽰转付费，禁 AI
⽣成/解析超时 loading 超时重试
图⽚违规 拦截提⽰
合规不通过 显⽰项+指引
导⼊格式不⽀持 提⽰⽀持格式
解析失败 重试/改⼿填
填充不完整 留空提⽰补
检测引擎故障 降级提⽰/⾃研兜底
⽤⼾取消 草稿暂存
存储/分享失败 重试/引导存相册
登录失效 静默重登
回执城市缺失 提⽰必填
证件照检测/抠图引擎⾮混元免费额度，成本独⽴评估（决策②）。
14. ⾮功能需求
性能：⽣图<20s、润⾊<15s、解析<25s、PDF<10s、⾸屏<2s。

可⽤性：AI 异步 100% 有反馈，⽆静默失败。
成本：额度监控、存储清理。
